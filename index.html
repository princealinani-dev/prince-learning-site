Week 1: Digital Foundations

Introduction to Computers and Basic Operations

Understanding Computer Systems

Computers are electronic devices that process data according to instructions provided by software programs. They consist of hardware (physical components) and software (programs and applications).

Key Learning Objectives

  • Identify basic computer hardware components (monitor, CPU, keyboard, mouse)
  • Understand the difference between hardware and software
  • Learn proper startup and shutdown procedures
  • Navigate the computer desktop interface
  • Create, save, and manage files and folders

Practical Activities

  1. Power on the computer and observe the startup process
  2. Practice using the mouse (click, double-click, right-click, drag)
  3. Explore the desktop interface and taskbar
  4. Create a new folder on the desktop
  5. Create a simple text document and save it in your folder

Key Terminology

Hardware: Physical components of a computer system (monitor, keyboard, CPU, etc.)

Software: Programs and applications that run on computer hardware

Operating System: Software that manages computer hardware and software resources (Windows, Linux, macOS)

Desktop: The main screen area you see after logging into your computer

Week 2: Word Processing Skills

Creating and Formatting Documents

Introduction to Word Processing

Word processing software allows you to create, edit, format, and print text documents. Common examples include Microsoft Word, Google Docs, and LibreOffice Writer.

Key Learning Objectives

  • Open and create new documents in word processing software
  • Type, edit, and format text (font, size, style, color)
  • Use paragraph formatting (alignment, spacing, indentation)
  • Insert and format images, tables, and other elements
  • Save documents in different formats and locations
  • Print documents with proper page setup

Practical Activities

  1. Create a new document and type a short paragraph about yourself
  2. Format the text with different fonts, sizes, and colors
  3. Practice aligning text (left, center, right, justified)
  4. Insert a table with your weekly schedule
  5. Add page numbers and save the document in PDF format

Formatting Tips

Consistency: Use the same font and formatting throughout your document

Readability: Choose fonts that are easy to read (Arial, Times New Roman, Calibri)

Spacing: Use appropriate line spacing (1.5 or double spacing for readability)

Headers/Footers: Use headers for document titles and footers for page numbers

Week 3: Internet Navigation

Safe Browsing and Online Research

Understanding the Internet

The Internet is a global network of computers that allows sharing of information and resources. Web browsers (Chrome, Firefox, Safari) are software applications used to access websites on the Internet.

Key Learning Objectives

  • Understand how the Internet works and basic terminology
  • Navigate websites using web browsers
  • Use search engines effectively to find information
  • Evaluate the credibility of online sources
  • Bookmark important websites for quick access
  • Understand browser security settings and features

Practical Activities

  1. Open a web browser and navigate to different educational websites
  2. Use a search engine to find information about a specific topic
  3. Bookmark three useful educational websites
  4. Practice using browser tabs to open multiple websites
  5. Clear browser history and cache for privacy

Search Engine Tips

Keywords: Use specific keywords rather than full sentences

Quotation Marks: Use "quotes" to search for exact phrases

Site Search: Use "site:" to search within a specific website (site:tanzania.go.tz education)

Evaluate Sources: Check the author, date, and credibility of information

Week 4: Online Safety

Cyber Security and Digital Citizenship

Understanding Online Threats

As we use the Internet more frequently, understanding online safety becomes crucial. This week focuses on protecting yourself from cyber threats and practicing responsible digital citizenship.

Key Learning Objectives

  • Create strong, secure passwords and manage them safely
  • Identify common online threats (phishing, malware, scams)
  • Understand privacy settings on social media and websites
  • Practice safe online communication and behavior
  • Recognize and avoid sharing personal information online
  • Understand copyright and ethical use of online content

Practical Activities

  1. Create a strong password using the guidelines provided
  2. Identify phishing emails from examples provided
  3. Review and adjust privacy settings on a social media platform
  4. Create a digital safety checklist for personal use
  5. Practice safe logout procedures on shared computers

Password Security Guidelines

Length: Use at least 12 characters

Complexity: Include uppercase, lowercase, numbers, and symbols

Uniqueness: Don't reuse passwords across different sites

Storage: Use a password manager or secure notebook (not digital files)

Sharing: Never share passwords with anyone

About the Program

Program Mission

This blended digital literacy program aims to bridge the technology gap for secondary school students in Tanzania by providing accessible, engaging, and practical ICT education. By combining traditional classroom instruction with mobile-accessible online learning, we ensure that students in rural areas can develop essential digital skills regardless of infrastructure limitations.

Mobile-First Learning Design

Students accessing educational content on smartphones, demonstrating adaptive technology use in resource-limited settings

Learning Methodology

Our approach is built on evidence-based educational practices, emphasizing collaborative learning, practical application, and incremental skill development. The curriculum is designed to be culturally relevant and context-appropriate for Tanzanian students, with content available in both Swahili and English.
